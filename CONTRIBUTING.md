# Contributing to flash-pkg

Thanks for your interest! Here's how you can help:

## Quick Contributions

- **Windows PowerShell version** — Most needed right now
- **More templates** — FastAPI, Django, Next.js, Svelte, etc.
- **Mirror improvements** — Add more Asia/Africa/LatAm mirrors
- **Bug fixes** — Test on your machine and report issues

## How to Contribute

1. Fork the repo
2. Create a branch: `git checkout -b feature/your-feature`
3. Make your changes
4. Test on your machine
5. Commit: `git commit -m "Add: your feature"`
6. Push: `git push origin feature/your-feature`
7. Open a Pull Request

## Testing

Before submitting:

```bash
# Test the install script
bash install.sh

# Test commands
flash ml test-project
flash mern test-app
```

## Code Style

- Keep it simple and readable
- Add comments for complex logic
- Follow existing patterns in `install.sh`

## Ideas Welcome

Open an issue if you have ideas for:
- New project templates
- Enterprise features
- Performance improvements
- Better mirror selection logic

Let's make dev setup instant for everyone! 🚀
